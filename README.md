#  An Autonomous Robot for Indoor Navigation Using A* Search for Path Planning While Avoiding Obstacles

An autonomous indoor mobile robot that uses the **A\*** search algorithm for optimal path planning while detecting and avoiding obstacles in real time. This project demonstrates the integration of robotics, artificial intelligence, and embedded systems to enable efficient and reliable indoor navigation.

---

#  Project Objective

The goal of this project is to develop an autonomous mobile robot capable of:

-  Mapping an indoor environment.
-  Planning the shortest path using the **A\*** search algorithm.
-  Detecting static and dynamic obstacles.
-  Avoiding obstacles during navigation.
-  Reaching the destination autonomously with minimal human intervention.

---

#  Features

- Autonomous indoor navigation
- A* path planning algorithm
- Real-time obstacle detection and avoidance
- Dynamic path replanning
- Differential drive robot control
- Sensor-based environmental perception
- Modular and scalable architecture

#  Expected Outcome

Upon completion, the robot will autonomously navigate an indoor environment by generating an optimal path, detecting obstacles in real time, adapting its route when necessary, and successfully reaching its destination without manual control. This project demonstrates the integration of intelligent path planning, robotic perception, and autonomous decision-making for practical indoor navigation applications.
